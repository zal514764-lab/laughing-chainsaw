# Arabic Sentiment Analysis Using Machine Learning

## Overview

This project focuses on Arabic sentiment analysis using machine learning techniques. The objective was to classify Arabic text into sentiment categories by applying natural language processing (NLP) preprocessing methods, feature extraction techniques, and supervised learning algorithms.

The project was developed as part of an Artificial Intelligence coursework project at Zarqa University.

## Dataset

* Approximately 56,000 Arabic text samples
* Sentiment-labeled data used for supervised learning
* Dataset preprocessed before model training

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* NLTK

## Project Workflow

### Data Preprocessing

* Text cleaning
* Tokenization
* Arabic text normalization
* Stop-word handling

### Feature Extraction

* TF-IDF Vectorization
* Character N-grams

### Machine Learning Models

* Logistic Regression
* Multinomial Naive Bayes
* Linear Support Vector Machine (SVM)

### Evaluation Metrics

* Accuracy
* F1 Score
* AUC-ROC

## Results

The models were compared using multiple evaluation metrics. The best-performing model was Linear SVM, achieving an accuracy of approximately 86.35%.
